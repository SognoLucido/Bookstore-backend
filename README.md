# Bookstore-backend
Simple backend api service for a bookstore

**Tools and Technologies** :
- ASP.NET Core Web API
- .net8
- efcore
- Mvc Controllers
- Minimal api (check branch)
- JWT with Asymmetric Key
- xunit
- Testcontainers
- SwaggerUI
- docker / compose
- Postgres
     
**Check Out the xUnit Tests** : [xUnit Tests](Tests/).
