Simple backend api service for a bookstore

- ASP.NET Core Web API
- .net8
- efcore
- Mvc Controllers
- Minimal api (check branch)
- JWT with Asymmetric Key
- xunit
- Testcontainers
- SwaggerUI
- docker / compose
- Postgres
